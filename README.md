# vim-config

My vim-config project

## Description
This repository holds my personal configuration files for Vim, designed to enhance productivity and streamline the editing experience.

## Features
- Custom keybindings
- Plugin management
- Color schemes

## Installation
To install, simply clone the repository and symlink the configuration files to your Vim directory:
```bash
git clone https://github.com/hauro-jeksysz/vim-config.git
ln -s vim-config/.vimrc ~/.vimrc
```
